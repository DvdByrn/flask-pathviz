Welcome to Metabolic Modeling Visualizer!

Metabolic Modeling Visualizer is a web-based tool for visualizing biological pathways. It is a great way to contextualize biological datasets.

Getting started

View pathway maps

  • Choose "Map" in the menu-bar above. This will open the Map Viewer with metabolic pathway maps for E.coli model iJO1366.

  • View, zoom and pan a map using the toolbar located on the left-hand side of the Map Viewer.

Change between different pathway maps

  • Switch between pathway maps using the drop-down located in the lower right-hand corner of the Map Viewer.

Load flux data onto maps

  • Flux data in CSV file-format can superimposed on a map using the Data Menu.

  • Click "Load flux data" and choose the CSV file to be uploaded.

  • To clear the flux data, click "Clear flux data" in the Data Menu.
